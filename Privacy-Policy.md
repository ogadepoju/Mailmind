# MailMind Privacy Policy

Last updated: February 2026

## Data Collected
MailMind accesses email content (subject line, sender name, and body text) 
solely to generate AI-powered reply drafts.

## How Data Is Used
Email content is sent to a secure backend server which forwards it to 
Anthropic's Claude API to generate a draft reply. This is the sole purpose 
of data collection.

## Data Storage
No email data is stored permanently. Data is processed in real-time and 
discarded after the draft is generated.

## Third Parties
Email content is shared only with Anthropic's Claude API for the purpose 
of draft generation. No data is sold or shared with any other third party.
